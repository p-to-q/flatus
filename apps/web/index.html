flatus

a small apparatus for moving air.

flatus is a desktop companion that lives in your menubar and occasionally farts. It is also, by acoustic accident, the same waveform Apple ships in watchOS to push water out of the Apple Watch speaker.

Make of that what you will.

# the CLI

$ git clone https://github.com/p-to-q/flatus
$ cd flatus
$ cargo install --path crates/fart-synth
$ fart                              # one shot
$ fart --personality biblical       # pick a voice
$ fart --render out.wav             # silent: write a WAV
$ fart --list-personalities

Not on crates.io yet. The desktop menubar app is the .app bundle — see the README's "First launch" for the right-click‑to‑Open dance.
