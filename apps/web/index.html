flatus

a small apparatus for moving air.

A desktop companion that lives in your menubar and occasionally farts. It is also, by acoustic accident, the same waveform Apple ships in watchOS to push water out of the Apple Watch speaker.

Download for macOS · Apple Silicon · v0.1.0 · unsigned try it in your browser ↓

First launch

  1. 1Your download is starting. The file is a ZIP — unzip it.
  2. 2Drag flatus.app into /Applications.
  3. 3Right-click flatus.app → Open. Gatekeeper will warn it's unsigned (we're working on it); click Open again.
  4. 4A small icon appears in your menubar. Left-click it to fart.

Right-click → settings (volume, personality, output, quiet hours). Quit from the same menu.

    Command line

    No menubar? No problem. The same synth ships as a CLI on every platform that has Rust.

    $ git clone https://github.com/p-to-q/flatus
$ cd flatus
$ cargo install --path crates/fart-synth
$ fart                              # one shot, default voice
$ fart --personality biblical       # pick a voice
$ fart --seed 42 --render out.wav   # write a WAV; identical to this page
$ fart --list-personalities
